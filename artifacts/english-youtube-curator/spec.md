# Spec: English YouTube Curator

## 개요

유튜브 알고리즘 없이 영어 수준에 맞는 영상을 탐색하고 앱 안에서 안전하게 시청할 수 있는 큐레이션 서비스. 아이가 레벨과 장르를 선택하면 검증된 영상 목록이 나타나고, 클릭 한 번으로 외부 이탈 없이 재생된다.

---

## 범위

### 포함

- 레벨 선택 화면 (Beginner / Intermediate / Advanced)
- 레벨 + 장르 조합으로 영상 썸네일 그리드 탐색 (최대 10개)
- 앱 내 YouTube 임베드 재생
- 영상 카드의 아동 친화 배지
- 관리자 페이지: 영상 추가 · 수정 · 삭제

### 제외

- 일반 사용자 로그인/계정 시스템 — MVP 범위 초과
- AI 자동 AR 분류 — "AR이 영상에 유효하다"는 가정이 검증되지 않음
- 재생목록 자동 생성 — 큐레이션 디렉토리 검증 후 추가
- 커뮤니티 기여 — 품질 관리 구조 없이 열면 신뢰 하락 위험
- 모바일 앱 — 웹 반응형으로 먼저 검증

---

## 시나리오

### 1. 레벨 선택 → 영상 목록 진입

- **Given** — 앱 홈 화면이 열려 있다
- **When** — 사용자가 "Beginner" 버튼을 클릭한다
- **Then** — Beginner 레벨에 해당하는 영상 썸네일 그리드와 장르 필터 탭이 표시된다

성공 기준:
- [ ] 앱을 열면 Beginner / Intermediate / Advanced 세 레벨 버튼이 화면에 표시된다
- [ ] 레벨 버튼 클릭 → 해당 레벨의 영상 목록 화면으로 전환된다
- [ ] 영상 목록 화면에는 장르 필터 탭 (전체 포함) 이 표시된다

---

### 2. 장르 필터 적용

- **Given** — Intermediate 레벨 영상 목록 화면이 열려 있고, 모든 장르의 영상이 표시되어 있다
- **When** — 장르 탭 "Song"을 클릭한다
- **Then** — "Song" 장르로 태깅된 영상 썸네일만 표시된다. 다른 장르의 썸네일은 사라진다

성공 기준:
- [ ] "Song" 탭 클릭 → Song 장르 영상만 표시된다
- [ ] "전체" 탭 클릭 → 해당 레벨의 모든 영상이 다시 표시된다
- [ ] 영상 목록은 어떤 필터 조합에서도 최대 10개를 초과하지 않는다

---

### 3. 앱 내 영상 재생

- **Given** — 영상 목록 화면에서 썸네일이 표시되어 있다
- **When** — 영상 썸네일을 클릭한다
- **Then** — 앱 화면 안에서 YouTube 영상이 임베드 플레이어로 재생된다. 브라우저가 YouTube 외부 페이지로 이동하지 않는다

성공 기준:
- [ ] 썸네일 클릭 → 앱 내 임베드 플레이어에서 영상이 재생된다
- [ ] 재생 중 브라우저 URL이 변경되지 않거나 앱 내 라우트 안에 머문다
- [ ] 재생 화면에서 영상 목록으로 돌아가는 버튼이 표시된다

---

### 4. 영상 카드의 아동 친화 배지

- **Given** — 영상 목록 화면에 영상 카드가 렌더링된다
- **When** — 해당 영상이 관리자에게 "Kids-friendly"로 표시된 경우
- **Then** — 영상 카드에 아동 친화 배지(아이콘 또는 텍스트)가 표시된다. Kids-friendly가 아닌 영상에는 배지가 없다

성공 기준:
- [ ] Kids-friendly 영상 → 카드에 배지가 표시된다
- [ ] Kids-friendly가 아닌 영상 → 카드에 배지가 없다

---

### 5. 관리자 — 영상 추가

- **Given** — 관리자 페이지가 열려 있다
- **When** — YouTube URL, 제목, 레벨, 장르, Kids-friendly 여부를 입력하고 저장 버튼을 클릭한다
- **Then** — 해당 영상이 사용자 화면의 영상 목록에 나타난다

성공 기준:
- [ ] 필수 필드(URL, 제목, 레벨, 장르)를 모두 입력하고 저장 → 영상이 목록에 추가된다
- [ ] 필수 필드 중 하나라도 비어 있고 저장 → 오류 메시지가 표시되고 저장되지 않는다
- [ ] 유효하지 않은 YouTube URL 입력 후 저장 → 오류 메시지가 표시된다

---

### 6. 관리자 — 영상 삭제

- **Given** — 관리자 페이지에 영상 목록이 표시되어 있다
- **When** — 특정 영상의 삭제 버튼을 클릭하고 확인한다
- **Then** — 해당 영상이 관리자 목록과 사용자 화면 양쪽에서 더 이상 표시되지 않는다

성공 기준:
- [ ] 삭제 확인 → 관리자 목록에서 해당 영상이 사라진다
- [ ] 삭제 확인 → 사용자의 영상 목록에서 해당 영상이 사라진다

---

## 불변 규칙

**외부 이탈 차단**: 앱 내 어디에서도 YouTube 외부 페이지로 이동하는 링크나 버튼이 없다. 모든 영상은 임베드 플레이어로만 재생된다.

**관리자 격리**: 관리자 페이지 URL은 일반 사용자 화면에서 링크로 노출되지 않는다.

**영상 수 제한**: 레벨 + 장르 조합으로 필터링된 영상 목록은 최대 10개까지 표시된다.

---

## 의존성

- YouTube IFrame Player API — 임베드 재생에 필요
- 채널/영상 큐레이션 데이터 최소 10개 이상 (관리자 페이지로 등록)

---

## 미결정 항목

- 레벨별 AR 수치 범위 (예: Beginner = AR 1.0~2.0) — 영어교육 전문가 검토 필요
- 장르 목록 확정 (Story / Song / Science / 기타를 기본값으로 제안하되 관리자가 추가 가능 여부)
- 관리자 인증 방식 (비밀번호 직접 입력 vs 환경변수 기반 접근)
