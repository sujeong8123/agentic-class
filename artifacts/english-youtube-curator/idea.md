# English YouTube Curator

## Problem Statement (HMW)

HMW — 유튜브 알고리즘에 아이를 맡기는 대신, 아이의 영어 수준에 딱 맞는 콘텐츠를 *먼저* 골라줄 수 있게 할 수 있을까?

> 틀 변경: 유해 콘텐츠를 막는 문제 → 좋은 콘텐츠로 채우는 문제

---

## Context

영어유치원에 다니는 아이(3~8세)가 유튜브를 볼 때, 알고리즘이 유해하거나 수준에 맞지 않는 콘텐츠를 추천한다. 현재 AR 지수별·나이별로 유튜브 채널/영상을 탐색할 수 있는 전용 서비스가 없다.

---

## 추천 방향: 큐레이션 채널 디렉토리

AR 1.0~5.0, 나이 3~8세 기준으로 수동 큐레이션된 영어 유튜브 채널 디렉토리를 먼저 만든다. 사용자 검증 후 AI 자동 분류를 추가한다.

### 핵심 기능 (MVP)

1. **채널 디렉토리** — AR 범위·나이·장르(스토리/노래/과학 등)로 필터링 가능한 채널 목록
2. **영상 추천** — 채널별 대표 영상 큐레이션, AR 수준 표시
3. **안전 표시** — 아동 친화 여부 배지 (Kids-friendly / 교육 채널 인증 여부)

### Not Doing (MVP)

- AI 자동 AR 분류 (검증 전까지 보류)
- 커뮤니티 기여 시스템
- 재생목록 자동 생성
- 모바일 앱

---

## 숨은 가정

| 분류 | 가정 | 검증 방법 |
|---|---|---|
| Must Be True | 부모/아이가 YouTube Kids 대신 이 앱을 실제로 선택한다 | 초기 사용자 인터뷰 or 랜딩 페이지 반응률 |
| Must Be True | AR 지수가 영어 영상 콘텐츠에 의미 있게 적용된다 | 샘플 영상 자막으로 AR 추정 후 전문가 검토 |
| Should Be True | 양질의 영어 학습 채널이 레벨별로 충분히 존재한다 | 사전 채널 리서치 (50개 목표) |
| Might Be True | AI 자동 분류 정확도가 실용 수준에 도달한다 | MVP 이후 실험 |

---

## 성공 기준

- 채널 50개 이상 큐레이션 완료
- AR 1.0~5.0 전 레벨 커버
- 사용자가 원하는 레벨의 채널을 3번 클릭 이내에 찾는다
